Practice 4: Diversity data set selection for sensitive groups' representations
Practice description

C2: Data preparation.

Edit the dataset according to the similarity and diversity of data and features to improve sensitive groups' representativeness and avoid discrimination among them (e.g. using clustering algorithms or ranking algorithms).

Application Example in Compas Scenario

Use a ranking or clustering algorithm to identify minority groups (more likely to be discriminated against) concerning skin color.
