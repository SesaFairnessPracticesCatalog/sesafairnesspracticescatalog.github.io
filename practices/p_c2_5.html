Practice 5: Causal analysis approaches to identify discrimination dependencies in data
Practice description

C2: Data preparation.

Use causal graphs approaches to identify and remove causes of discriminations from datasets, balancing features, utility and fairness of data.

Application Example in Compas Scenario

Use manual or automatic causal graph generation approaches for the COMPAS model to assess any dependence between model results and sensitive features, such as skin color.
