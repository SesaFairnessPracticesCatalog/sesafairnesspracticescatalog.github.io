Practice 2: Focused learning strategies to obtain discrimination-free outcomes
Practice description

C3: Model building.

Use focused machine learning strategies (clustering-based learning algorithms, for instance) that satisfy specific fairness definitions and constraints to obtain discrimination-free outcomes balancing dependencies between sensitive and not sensitive features.

Application Example in Compas Scenario

Use learning approaches based on clustering analysis in order to produce targeted output results for each minority group identifiable by skin color.

Addictional considerations

The application of focused learning strategies to obtain discrimination-free outcomes was per- ceived as having high impact on the fairness of a ML-Intensive system (with 17 preferences). This practice requires an average or high effort to be applied (15 average and 17 high opinions). Conse- quently, practitioners observed how this practice is averagely put in practice (with 19 opinions).
